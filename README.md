## 📖 Overview

This project is a web-based precision agriculture application developed as part of a university team project. It leverages machine learning models trained on agricultural datasets to:

- 🌱 Recommend suitable crops based on environmental conditions.
- 🌾 Predict crop yield using agricultural parameters.
- 💻 Provide an easy-to-use web interface built with Flask.

The project combines crop recommendation and crop yield prediction into a single application, making agricultural insights more accessible through data-driven decision-making.
---

## ✨ Features

- 🌱 **Crop Recommendation** based on soil and environmental parameters.
- 🌾 **Crop Yield Prediction** using machine learning models trained on agricultural datasets.
- 🖥️ **Interactive Flask Web Interface** for easy user interaction.
- 📊 **Data-Driven Predictions** using preprocessed agricultural datasets.
- 📈 **Machine Learning-Based Decision Support** to assist users in selecting suitable crops.
- 🧩 **Unified Application** combining crop recommendation and crop yield prediction within a single platform.
- 🎯 **Simple and User-Friendly Interface** designed for quick and intuitive usage.
